Desenvolvimento Web com HTML

Atualmente, praticamente todos os sistememas corporativos possuem interfaces web. Para quem deseja atuar no mercado de desenvolvimento de software, é obrigatório o conhecimento das linguagens: HTML, CSS e JavaScript

Essas linguagens são utilizadas para o desenvolvimento da cama de apresentação das aplicações web.

Pré-requisitos

  • Familiaridade com algum sistema operacional
  • Lógica de programção
